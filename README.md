# Containers (WIP)

### Configuration

You can configure your project with a `.containers.yml` file.
This file can be generated by running: `containers generate config`

```yaml
---
organization:
  name: my-organization

app:
  name: my-project
  directory: .

docker:
  directory: .
  default_service:
  compose_files:
    - docker-compose.yml
```
